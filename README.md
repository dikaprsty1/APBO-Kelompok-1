## Judul Project

**Smart Kost Management System: Aplikasi Pengelolaan Kost Berbasis Web dengan Notifikasi dan Tracking Pembayaran**

---

## Analisis 


---

## Problem

* Pendataan penyewa masih dilakukan secara manual sehingga rawan kesalahan dan kehilangan data
* Sulit memantau status ketersediaan kamar secara real-time
* Pengelolaan pembayaran belum terorganisir dengan baik
* Tidak adanya sistem pengingat jatuh tempo pembayaran bagi penyewa
* Kesulitan dalam membuat laporan pendapatan dan data penghuni secara otomatis
* Komunikasi antara penyewa dan pemilik kost belum terintegrasi dalam satu sistem

---

## Tujuan

* Membuat sistem untuk mengelola data penyewa secara terstruktur
* Menyediakan fitur monitoring ketersediaan kamar
* Mengelola dan mencatat pembayaran secara digital
* Memberikan notifikasi jatuh tempo pembayaran
* Menghasilkan laporan secara otomatis dan akurat

---

## Fitur Utama (Rencana)

* Manajemen data penyewa
* Manajemen kamar (kosong/terisi)
* Sistem pembayaran dan riwayat transaksi
* Notifikasi jatuh tempo pembayaran
* Dashboard admin
* Laporan bulanan
